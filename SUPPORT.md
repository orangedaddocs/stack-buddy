# Support

Stack Buddy is a personal open-source MVP.

I am sharing it because it may be useful to other people, but I am not promising active maintenance, support, bug fixes, feature requests, or hosted uptime.

If you find a bug, you can open an issue.

If you improve the project, feel free to fork it or open a pull request.

Please do not send private financial information, seed phrases, wallet details, tax documents, or personally identifying financial data.

This project should never ask for your seed phrase.
